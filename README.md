# 🚀 Performance Marketing Automation (Claude AI Edition)

A professional-grade toolkit for scaling Google Ads accounts using **Claude 3.5 Sonnet** and automated scripts. This repository focuses on bridging the gap between raw data and high-intent creative execution.

## 🤖 Claude AI Integrations
* **Creative Auditing:** Automated analysis of ad headlines and descriptions against brand guidelines.
* **Search Term Intent Mapping:** Using Claude to categorize thousands of search terms into "High Intent," "Informational," or "Negative."
* **Landing Page Analysis:** Scripts that send landing page HTML to Claude for conversion rate optimization (CRO) suggestions.

## 🛠 Features
- **Dynamic Scripting:** Google Ads scripts that pull real-time data for AI processing.
- **Reporting:** Automated Looker Studio connectors with AI-generated summaries.
- **Scaling Logic:** Automated bid adjustments based on performance trends.

## 📂 Project Structure
- `/scripts`: Raw Google Ads JavaScript files.
- `/prompts`: Optimized Claude system prompts for marketing analysis.
- `/automation`: n8n/Make.com JSON blueprints.

## 🚀 Getting Started
1. Get your **Anthropic API Key**.
2. Deploy the `search-term-analyzer.js` script to your Google Ads account.
3. Connect the webhook to your Claude automation workflow.
